Database Setup Instructions

To set up the database:

  1. Open phpMyAdmin from Laragon
  2. Create a new database named "itemtrack"
  3. Select the "itemtrack" database
  4. Go to the "Import" tab
  5. Click "Choose File" and select the itemtrack_db.sql file
  6. Click "Go" to import the database structure and sample data
